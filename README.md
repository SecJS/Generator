## Installation

First, install [Yeoman](http://yeoman.io) and SecJS Generator using [npm](https://www.npmjs.com/) (we assume you have pre-installed [NodeJS](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-secjs
```

Then generate your files:

```bash
yo secjs
```

or

```bash
yo generator-secjs
```
